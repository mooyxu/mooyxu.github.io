Markdown

简介：markdown是一门轻量级的语言，用来记录文本信息：文章、笔记等。它比单纯的txt文本拥有更多的内容格式，却不像html/xml那样有复杂且不直观的语法。现在它已经被广泛使用，且能对其他的复杂格式转化：html、pdf等。

语法简介：markdown分为区块元素和区段元素，对应html中的块级元素(block)和行级元素(inline)。

  1. 区块元素有段落、标题、区块引用、列表、代码区块、分隔线。
  2. 区段元素有链接、强调、代码、图片。
  3. 其他还有反斜杠、自动链接等。
